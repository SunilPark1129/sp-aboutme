## Project Description

The 'About Sunil' website is a 3D project using React Three Fiber to showcase information about me.

## Installation
1. Clone the repo
```
git clone https://github.com/SunilPark1129/sp-aboutme.git
```
2. Install
```
npm install
```
2. run the website
```
npm run dev
```
[Link](https://sp-aboutme.netlify.app/)
